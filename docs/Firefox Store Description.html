Unofficial extension for Omnivore users who want to access their Omnivore items directly in a little popup. This extension is heavily inspired by the original In My Pocket extension for Pocket. Setup:
  1. Get an Omnivore API Key through your Omnivore Account.
  2. Install the extension from the Firefox Extension store.
  3. Go into the extensionâ€™s settings and paste your Omnivore API key.
  4. Launch the extension by clicking the new "Omnivore List Popup" button in your Firefox extension toolbar/dropdown.
  5. Optional: Add the extension to your always-visible toolbar.
