Unofficial extension for Omnivore users who want to access their Omnivore items directly in a little popup. This extension is heavily inspired by the original In My Pocket extension for Pocket. Features:
  • See all your Omnivore articles in a popup
  • Quickly open the original article URL in a new tab by clicking the article
  • Keep the popup open by keeping CMD (CTRL on Windows/Linux) pressed while clicking an article
  • Quickly archive the article with the archive button
  • Change the labels of an article
  • Open the Omnivore main website
  • More features coming soon!
Setup:
  1. Get an Omnivore API Key through your Omnivore Account.
  2. Install the extension from the Firefox Extension store.
  3. Go into the extensionâ€™s settings and paste your Omnivore API key.
  4. Launch the extension by clicking the new "Omnivore List Popup" button in your Firefox extension toolbar/dropdown.
  5. Optional: Add the extension to your always-visible toolbar.
